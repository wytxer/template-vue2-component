# Template Vue2 Component

基于 Vue.js 2.x 版本且风格统一的用于快速构建组件的脚手架。


## 快速使用

### 安装

```bash
# npm
npm i

# yarn
yarn install
```

### 使用

```bash
# 开发
npm run dev

# 打包
npm run build

# 开发组件
npm run dev:lib

# 打包组件
npm run build:lib
```
